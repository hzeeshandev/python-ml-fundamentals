## 🚀 Getting Started (Running This Repo Locally)

Clone the repository:
```bash
git clone git@github.com:hzeeshandev/python-ml-fundamentals.git
cd python-ml-fundamentals
```

Set up a virtual environment:
```bash
python -m venv venv
source venv/bin/activate      # Mac/Linux
venv\Scripts\activate         # Windows
```

Install dependencies (once a `requirements.txt` exists):
```bash
pip install -r requirements.txt
```

## 📖 Learning Roadmap

This repo follows a structured curriculum:

1. **Core Python** — syntax, data types, control flow, functions, error handling, file I/O
2. **OOP & Intermediate Python** — classes, inheritance, decorators, generators, regex
3. **Developer Tools** — Git workflows, unit testing, working with APIs, Jupyter notebooks
4. **Data Science** — data manipulation and visualization with NumPy, Pandas, Matplotlib, Seaborn
5. **Machine Learning** — supervised/unsupervised learning, model evaluation, tuning with scikit-learn
6. **Deep Learning** — neural networks, CNNs, intro to transformers, TensorFlow/PyTorch
7. **Applied Development** — REST APIs, databases, web scraping, deployment
8. **Capstone Projects** — end-to-end projects combining all of the above
